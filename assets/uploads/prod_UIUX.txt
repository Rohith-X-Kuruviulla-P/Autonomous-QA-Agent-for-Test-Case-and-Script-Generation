UI/UX Design Standards

1. Color Palette
   - Primary Success Action ("Pay Now"): Must be Green (#28a745).
   - Error Messages: Must be displayed in Red text for high visibility.
   - Success Message: Must be displayed in a light green box with dark green text.

2. Interaction
   - Hover Effects: Buttons should darken slightly when hovered to indicate interactivity.
   - Validation Feedback: Error messages should appear immediately below the invalid input field.

3. Typography
   - Total Price: Should be displayed in bold font to distinguish it from other text.
